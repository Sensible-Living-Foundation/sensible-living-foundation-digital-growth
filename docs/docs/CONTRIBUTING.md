# Contributing to Sensible Living Foundation Digital Growth

Thank you for helping improve the Sensible Living Foundation website and digital systems.

## How to contribute

1. Review open issues
2. Choose a task
3. Comment that you want to work on it
4. Complete the task
5. Submit your work

## Contribution areas

- UX/UI
- SEO
- automation planning
- content writing
- documentation
- website improvements

## Goal

This project helps improve the foundation while giving contributors real nonprofit project experience.
